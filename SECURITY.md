# Security Policy

## Reporting

Report suspected vulnerabilities or exposed sensitive material to:

<support@genesisre.io>

Do not open public issues containing secrets, private prompts, raw logs,
customer files, wallet keys, payment credentials, tenant traces, or exploit
payloads that expose non-public information.

## Public MCP Data Boundary

The public ProofRelay MCP endpoint accepts only non-confidential or synthetic
evidence metadata. Submit hashes, receipt identifiers, schema-safe metadata,
and public-safe fixtures only.

Do not submit:

- secrets or API keys
- private prompts
- source code
- raw customer data
- raw logs or telemetry traces
- wallet keys or payment credentials
- confidential documents or regulated records

## Non-Claims

ProofRelay does not provide payment processing, wallet custody, escrow, legal
advice, title certification, compliance certification, model-safety
certification, or real-world-truth attestation.
